This is a list of Object Oriented Terms and Definitions

Class
An abstraction that encapsulates data and behavior. A class is and defines the attributes that an instance of that class (an object) will have and the methods that it will respond to
Class Definition
Written code that describes a class
Instance or Object
A unique copy of a class that exists only while the program is running
Inheritance
A way to create a class definition that borrows code from another class
Class Hierarchy
A way to describe the ordering among classes using terms such as subclass, superclass, etc.
Method
A command that returns data from, or performs an action, an object
Attribute or Property
Data that can be set or retrieved from the object
Variable
A name that can be assigned to a value or object
Expression or Statement
An combination of variables, methods, and classes that returns a result
